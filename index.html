Cómo usar en Blender:
1. Abrir Blender → pestaña Scripting
2. Clic en New → pegar el código → Alt+P
3. Exportar: File → Export → STL
4. Verificar: Scene Properties → Unit Scale = 0.001, Length = Millimeters
